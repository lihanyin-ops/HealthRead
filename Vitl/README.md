# Vitl iPhone + Apple Watch

SwiftUI 双端工程，依据 `vitl-prototype` web 原型、PRD 和开发技术文档还原。

## 还原范围

- iPhone：首页、洞察、历程、设置四个 Tab。
- Apple Watch：能量、健康数据、AI 建议三页分页。
- 共享：健康快照模型、7 天 mock 数据、液体能量模型、能量颜色映射、订阅状态、基础用户偏好。
- 骨架：HealthKit 授权、AI Prompt/摘要服务、StoreKit 2 订阅管理占位。

## 打开方式

用 Xcode 打开 `Vitl.xcodeproj`，运行 `Vitl` 或 `Vitl Watch App` scheme。
